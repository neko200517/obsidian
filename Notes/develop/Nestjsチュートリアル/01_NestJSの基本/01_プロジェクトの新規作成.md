## 公式URL

https://docs.nestjs.com/

## NestCLIのインストール

```bash
npm i @nextjs/cli
```

### 代替

githubのテンプレートをクローンする方法もある

```bash
git clone https://github.com/nestjs/typescript-starter.git project
```

## プロジェクトの作成

グローバル環境にインストールしないため npx を使用する

```bash
npx nest new project-name
```
