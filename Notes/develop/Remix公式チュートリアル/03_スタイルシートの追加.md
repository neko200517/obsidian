## スタイルシートをインポートする

- linksにスタイルシートを複数インポートでき、この内容はLinkコンポーネントで展開される。

### app/root.tsx

```tsx
import type { LinksFunction } from "@remix-run/node";
// existing imports

import appStylesHref from "./app.css?url";

export const links: LinksFunction = () => [
  { rel: "stylesheet", href: appStylesHref },
];
```
