## サーバーの起動

```bash
npm run dev
```

## 動作確認

1. ブラウザを二つ立ち上げ、localhost://4000/graphql を開く 
2. 受信側を実行する 

### Operation

```json
subscription {
  newLink {
    id
    url
    description
    postedBy {
      id
      name
      email
    }
  }
}
```

3. 送信側を実行する

### Operation 

```json
mutation Post {
  post(url: "https://yahoo.co.jp", description: "Yahooサイト") {
    id
    description
    url
  }
}
```

1. 以下のようなレスポンスが返ってきたらOK

### Response

```json
{
  "data": {
    "newLink": {
      "id": "50",
      "url": "https://yahoo.co.jp",
      "description": "Yahooサイト",
      "postedBy": {
        "id": "1",
        "name": "user1",
        "email": "hoge@gmail.com"
      }
    }
  }
}
```
